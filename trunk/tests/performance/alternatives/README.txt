Steps to build the alternatives performance test:

1) Download Robert Penner's as3signals from here:
   http://github.com/robertpenner/as3-signals

2) Extract the source code from the ZIP or TAR.GZ file

3) Copy as3signals' src/org directory to Turbo Signals'
   tests/performance/alternatives/src directory

4) Build the alternatives performance test:
   ant alternativesPerformanceTests
